The third group assignment of the Principles of Info. Systems course is named "How websites work?". The goal of the assignment is to build a website using HTML, CSS, and Javascript as the technologies, and track the changes on a popular source code hosting website, Github using a version control system tool. Utilization of ready templates and tools except Visual Studio Code is prohibited. The website has to also include no less than 5 web pages, with one dedicated to the creation of art using algorithms. The deadline of the project is December 1st, 2024. This assignment tests the team's knowledge of the common web technologies learnt from the assignment that was published prior to the current one.
Prior to commencing the assignment, the team leader, Elmar Verdiyev, made important design choices. The final decision included the use of "Poppins" as the main font of the website. Furthermore, the color pallete was decided as: #00ADB5, #222831, #EEEEEE, and #393E46. The work of individual members was constantly checked to ensure consistency on all 5 web pages of the website. The checks included ensuring matching fonts and colours were used relative to the initial standards. The only web page greenlighted to ignore the standards minimally was the Hour of Code webpage. This decision was made due to the creator's ability to harmonize, similar to other web pages, with the use of carefully choosen font and design.
